Buttons

Buttons are used to perform any action by clicking on it. It makes the web page interactive to the user.

Solid Buttons

Solid buttons have more emphasis, as they use a color fill. They contain actions that are primary to your app.

You can add these buttons by just using "btn and btn-text" class along with it's btn-variant.

Outlined Buttons

Outlined buttons are medium-emphasis buttons. They contain actions that are important, but arenâ€™t the primary action in an app.

You can add these outlined buttons by just replacing btn-variant with btn-variant-outline.

Link Button

Do you want to attach a link action at a button tap? Then use Link Button.

You can add this link button by adding "btn-info" class along with "btn and btn-text" class.

Icon Button

Icon Buttons can be created by using an image as a child to parent button.

You can add this icon button by adding "icon-button" class along with "btn, btn-text and btn-info" class.

Floating Action Button

A floating action button performs the primary, or most common, action on a screen.
