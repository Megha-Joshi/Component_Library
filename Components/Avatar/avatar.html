Avatar

Avatars are images that users can set as their profile picture, usually found in circular shapes.They can be custom photos or uploaded by users.

Simple Round Avatar

Simple round avatars are circular in shape and used by the users in their profile pictures.

You can use round avatar by adding "round" class with the common class "avatar-image".

Sizes Avatars

Size of the avatars can be changed by adding different sizes to the existing avatar class.

You can add different sizes avatar by using sm|lg to the common class "avatar-image".
