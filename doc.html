Introduction

Sparrow is an open source styling framework, which helps Developers to use best styles and components, by using pre-defined classnames. Sparrow helps you to quickly build your Dream Project, by focusing much on Functionality, because we take care of your Designs.

Customizing Themes

Storm aims at the satisfaction of user as much as possible but there are times when the pre-defined styles are not sufficient. In order to override Storm's base classes, user can add there own class to the pre-existing classes, and style them seperately.
